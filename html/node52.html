Analysis of results

This data is subject to human interpretation - there is no hour by hour detail only nightly totals - there is also no combination data (when bad weather and technical downtime occur simultaneously). There is also a bias such that when combinations do occur this is logged as bad weather. The plots do reveal a tendency for better weather in summer and more bad weather in winter as might be expected but beyond that little of use in prediction - plots of run lengths (consecutive days where fraction of time lost to bad weather exceed given thresholds are shown in figure 5.5.5 - similar to collected WMS data.

Using this plot one can predict the likely length of a current run of bad weather based on the length upto the present time using bayes theorem E.g. if the current run is 2 days long, the probability of the run going on for another 12 days or more is P(14)/P(2) = 0.43.

Figure 28: The figure shows the probability of the length of a run of continuous bad weather for exceding given threshold values. Using this plot it is possible to determine the the likely length of the current bad weather run based on the length of run to-date.
