Long period comparison

Aim is to test a series of ODS with different selection, scoring and biasing then some basic LAS with different placement heuristics over a period of 2 months (2 lunar cycles) to determine tendenciesdies. A common environmental scenarionis kept in place throughout all tests. The following schedulers/components are to be used:-

  • ODS using standard HSMs (name for this ?).
  • Same but with different bias models i.e fixed rank bias, relative score bias.
  • Change to use new HSMs (name for this ?).
  • And with biasing and ranked metric selection (this is the multiple-queued thing).
  • Switch to simple LAS and test placement strategies.

Later we will fiddle with these and use different types of E and W prediction models (ie with different accuracy though not real models.) to see what works best under different scenarios - i.e want to test these under varying scenario parameter as independant variable(s).

Figure 11 shows a snapshot of the environmental (seeing) scenario used for this study. It consists of some 1800 seeing category changes over the 60 day period (average duration 48 minutes) with individual durations taken from the seeing model described in table 11 and photometric/spectrometric extinction ratio set to 41.

before we start need some idea of what sort of P2M to use - real or generated. A real model suffers from the ODB evolution problem (see ). With a generated P2 model, group activations are spread out over a specified period (say several months) so new groups are appearing in the schedule every day - like the real ODB but in that case they are not entered until required so we dont know about them in advance and cannot take them into account easily.

First rset up a few generated P2 models and run some tests - basically use a simple ODS with the planned environmental scenario and see how some easily calculated and visualized PCM looks - the daily average value of the dynamic contention profile is chosen as it is easily visualized. These tests are all run from 15-oct-07 to 15-dec-07 and have different levels of contention. Table 11 shows details of the phase2 models tested.
